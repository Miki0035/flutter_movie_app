## <a name="quick-start"> ğŸ¤¸ Quick Start </a>

Follow this steps to setup the project locally on your machine.

**Prerequsites**

Make sure you have the following installed on your machine

- [Git](https://git-scm.com/)
- [Dart](https://dart.dev/get-dart)
- [Flutter](https://docs.flutter.dev/get-started/)

**Cloning the Repository**

```bash
git clone https://github.com/Miki0035/flutter_movie_app

cd flutter_movie_app
```

**Check everything**

Check if everything is setup to run a flutter application:

```bash
flutter doctor
```

**Installation**

Install the project dependencies using flutter:

```bash
 flutter pub get
```

**Enviroment Variables**

Create a .env file at the root of your project and add the variables as specified in the .env.example file

```bash
 flutter pub get
```

**Running the Project**

Install an android emulator or use your android device in debug mode

```bash
flutter run
```
